# 更新日志

## 2026-02-23 - 文档系统优化

### 新增功能

#### 1. 文档语言自动跟随网站语言
- ✅ 文档语言现在会自动跟随顶部导航栏的语言切换
- ✅ 如果网站设置为中文，文档自动显示中文版本
- ✅ 如果网站设置为英文，文档自动显示英文版本
- ✅ 移除了文档页面独立的语言切换按钮，统一使用顶部导航栏的语言切换

**实现方式：**
```typescript
// 使用 i18n 的语言设置
const { i18n } = useTranslation();
const lang = i18n.language.startsWith('zh') ? 'zh' : 'en';
```

#### 2. Markdown 内部链接自动跳转
- ✅ 点击文档内的 `.md` 链接会自动跳转到对应文档
- ✅ 无需刷新页面，平滑切换文档
- ✅ 自动滚动到页面顶部
- ✅ 外部链接（http/https）在新标签页打开

**支持的链接格式：**
```markdown
[什么是 blockcell？](./01_what_is_blockcell.md)
[5分钟上手](./02_quickstart.md)
[GitHub](https://github.com/blockcell-labs/blockcell)
```

**实现逻辑：**
1. 检测链接是否以 `.md` 结尾
2. 提取文件名中的序号（如 `01_what_is_blockcell.md` → `01`）
3. 根据序号切换到对应文档
4. 平滑滚动到页面顶部

3. 文档页面自动切换到对应语言的文档

4. 自动滚动到顶部

### 已知限制

1. **锚点链接暂不支持**
   - 如 `[跳转到某节](#section)` 暂时不会自动处理
   - 可以在后续版本中添加

2. **相对路径必须以 `./` 开头**
   - 正确：`./01_what_is_blockcell.md`
   - 错误：`01_what_is_blockcell.md`

3. **文件名必须符合命名规范**
   - 格式：`XX_title.md`（XX 为两位数字）
   - 如果不符合规范，链接跳转可能失败
