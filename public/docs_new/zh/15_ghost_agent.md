# 幽灵智能体（Ghost Agent）

> 在后台默默工作的守护者——整理记忆、主动提醒、监控系统健康

---

## 什么是 Ghost Agent？

blockcell 里有一个特殊的组件叫做 **Ghost Agent（幽灵智能体）**，之所以叫"幽灵"，是因为它：

- 在**后台**悄悄运行，你通常感受不到它的存在
- 但它一直在**观察**你的对话和系统状态
- 偶尔会**主动发声**——提醒你重要的事情，或者问候一声

类比：就像一个贴心的秘书，你在开会（主对话），而他在旁边默默整理会议记录、提前安排下一件事，偶尔递上一张便利贴提醒你。

---

## Ghost Agent 做什么？

Ghost 定期（默认每 60 秒）在后台执行一次**例行检查**，处理以下四类任务：

### 1. 记忆管理（memory）

回顾最近的对话历史，提取值得长期保存的信息并保存到记忆库。

例如：
- 你提到了一个新的项目，Ghost 会记住项目名称和关键信息
- 你设置了一个偏好（"我喜欢简洁的代码"），Ghost 确保它被保存
- 你的持仓发生了变化，Ghost 更新记忆

### 2. 记忆清理（cleanup）

清理过时、冗余或已过期的短期记忆，保持记忆库干净整洁。

例如：
- 昨天的"明天开会"提醒已经过期，可以删除
- 同一主题的重复记忆，合并为一条更准确的记录

### 3. 主动消息（social）

如果有需要，Ghost 会主动发一条消息给你。这不常发生，但在某些情况下很有用：

- 你好久没有交互了，而系统检测到有重要事项需要提醒
- 某个监控指标刚刚触发（如股价告警），Ghost 确保你收到通知
- 技能进化完成，需要告知你结果

### 4. 问题检测（issues）

检查系统是否有需要关注的问题：

- 某个技能频繁失败，是否需要进化？
- 告警规则是否工作正常？
- 记忆库是否过大，需要维护？

---

## Ghost 的输出格式

Ghost 以结构化 JSON 格式输出它的判断，确保高效且不浪费 token：

```json
{
  "memory": [
    {
      "action": "upsert",
      "title": "用户茅台持仓更新",
      "content": "用户持有茅台 500 股，成本价 1820 元，2025-01-15 更新",
      "type": "fact",
      "importance": 8
    }
  ],
  "cleanup": ["dedup_key_old_maotai_position"],
  "social": null,
  "issues": []
}
```

如果没有需要处理的事情，所有字段都是空的，Ghost 保持沉默。

## Ghost 与主对话的关系

Ghost 和你的主对话是**完全独立**的：

- 主对话处理你的实时请求
- Ghost 在后台异步运行，不会打断主对话
- Ghost 的行动（保存记忆、发消息）通过标准工具执行，有完整的审计追踪
- Ghost 是**轻量**的：它的系统提示从约 800 字符压缩到约 300 字符，每次运行消耗很少的 token

---

## Ghost 与自我进化的关系

Ghost Agent 是自我进化系统的**观察者和催化剂**：

1. Ghost 持续观察各技能的运行情况
2. 如果发现某个技能有改进空间（即使还没达到自动进化阈值），Ghost 可以提前标记
3. Ghost 也会观察进化完成的效果，确保改进真的有效

这形成了一个完整的**AI 自我维护闭环**：
```
执行 → 观察（Ghost）→ 记忆 → 改进（Evolution）→ 再执行
```

## 配置 Ghost Agent

Ghost 默认是启用的，你可以在 `config.json` 中调整：

```json
{
  "agents": {
    "ghost": {
      "enabled": true,
      "intervalSecs": 60,
      "model": "deepseek-chat"
    }
  }
}
```

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `enabled` | `true` | 是否启用 Ghost |
| `intervalSecs` | `60` | 运行间隔（秒） |
| `model` | 主模型 | Ghost 使用的模型（可以用便宜的小模型） |

**建议**：Ghost 的任务相对简单，可以配置成本较低的模型（如 `deepseek-chat`），节省费用。

## 为什么叫"幽灵"？

因为一个好的助手是**无形的**：

- 你不用管它，它自己做该做的事
- 它不会打扰你，除非真的需要
- 但当你需要的时候，它留下的"痕迹"（记忆、提醒）让你感到它一直都在

就像电影里那些守护家宅的温和幽灵——存在感很弱，但房子因为它们变得更好。
