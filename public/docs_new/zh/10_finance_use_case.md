# 金融场景实战

> 用 blockcell 搭建你的专属金融 AI 助手

---

## 概览

blockcell 内置了完整的金融数据能力，可以帮你：

- 实时监控 A 股 / 港股 / 美股 / 加密货币
- 自动生成每日 / 每周金融报告
- 设置价格告警，触发时自动通知
- 分析链上数据、DeFi 协议
- 构建量化策略原型

本文用几个真实场景演示如何使用。

---

## 场景一：A 股实时行情查询

### 基本查询

直接说股票名称或代码：

```
你: 茅台现在多少钱？

你: 查一下 601318 的行情

你: 腾讯和阿里今天分别涨了多少？
```

blockcell 会自动识别 A 股（6位数字）、港股（5位数字或 .HK 后缀）、美股（英文 ticker），选择对应的数据源：
- **A 股**：东方财富 API（免费，实时）
- **港股**：东方财富香港 API
- **美股**：Alpha Vantage + Yahoo Finance 备用

### 技术指标分析

```
你: 帮我看一下茅台最近 60 天的 K 线，计算 MA5、MA20、MACD 和 RSI

你: 平安银行现在处于什么技术形态？有没有买卖信号？
```

AI 会：
1. 获取 K 线历史数据（`finance_api stock_history`）
2. 在本地计算技术指标（无需额外 API）
3. 分析技术形态并给出判断

### 深度数据

```
你: 查一下茅台今天的资金流向

你: 看看平安银行最近的龙虎榜数据

你: 现在北向资金流入最多的是哪些股票？
```

这些数据通过 `http_request` 工具直接调用东方财富公开 API。

## 场景二：价格告警

### 设置告警

```
你: 帮我设置一个告警：茅台跌破 1500 元时，发 Telegram 通知给我

你: 比特币涨超 10 万美元时给我发消息

你: ETH 24 小时涨幅超过 15% 时提醒我
```

AI 会调用 `alert_rule` 工具创建告警规则：

```json
{
  "name": "茅台跌破 1500",
  "source": "finance_api",
  "symbol": "600519",
  "field": "price",
  "operator": "lt",
  "threshold": 1500,
  "on_trigger": [
    {
      "tool": "message",
      "params": {
        "channel": "telegram",
        "content": "⚠️ 茅台（600519）已跌破 1500 元，当前价格：{value}"
      }
    }
  ]
}
```

### 复杂告警条件

```
你: 帮我设置：当茅台 RSI 跌破 30（超卖），同时成交量比 20 日均量放大 2 倍以上时，提醒我可能有反弹机会

你: 当 BTC 的恐贪指数低于 20 时，自动记录到我的笔记里
```

### 查看和管理告警

```bash
blockcell alerts list
blockcell alerts delete "茅台跌破1500"
```

## 场景三：自动日报

### 设置每日晨报

```
你: 帮我设置一个定时任务，每天早上 8:30 自动生成 A 股开盘前的简报，发到我的 Telegram

简报要包含：
1. 昨日沪深指数涨跌
2. 资金流向（主力净流入前 5 行业）
3. 北向资金
4. 我的自选股（茅台、平安、宁德时代）的最新情况
```

AI 会：
1. 创建 cron 任务（每天 08:30 执行）
2. 编写一段执行逻辑，调用相应数据 API
3. 格式化为易读的报告
4. 通过 Telegram 发送给你

### 加密货币日报

```
你: 每天晚上 10 点给我发一份加密市场日报，包含 BTC/ETH/SOL 价格、24h 涨跌、大盘总市值、主要 DeFi 协议 TVL 变化
```

## 场景四：投资组合追踪

### 初始化持仓

先告诉 AI 你的持仓（AI 会记住）：

```
你: 记住我的股票持仓：
- 贵州茅台（600519）：500 股，成本价 1820 元
- 中国平安（601318）：2000 股，成本价 44.5 元
- 宁德时代（300750）：100 股，成本价 230 元

记住我的加密持仓：
- BTC：0.5 个，均价 90000 USDT
- ETH：5 个，均价 3200 USDT
```

### 实时盈亏查询

```
你: 帮我算一下我现在的股票持仓总盈亏

你: 我的加密资产现在值多少？和昨天比变化了多少？
```

AI 会获取最新价格，结合记忆中的持仓信息计算盈亏。

### 周度复盘

```
你: 帮我做一个本周投资组合的复盘报告，包括：
- 各仓位涨跌情况
- 对比大盘表现
- 最大回撤和当前浮盈
- 下周需要关注的关键事项
```

## 配置 Alpha Vantage API Key（可选）

用于美股数据和外汇数据。Alpha Vantage 提供免费 Key（每分钟 5 次请求）：

1. 注册：[https://www.alphavantage.co/support/#api-key](https://www.alphavantage.co/support/#api-key)
2. 获取免费 API Key
3. 配置：

```json
{
  "providers": {
    "alpha_vantage": {
      "apiKey": "你的AlphaVantage Key"
    }
  }
}
```

没有 Alpha Vantage Key 也可以，会自动降级使用 Yahoo Finance。

---

## 配置 CEX 交易所 API（可选）

如果需要查询真实账户余额或下单：

```json
{
  "providers": {
    "binance": {
      "apiKey": "你的Binance API Key",
      "apiBase": "你的Binance API Secret"
    }
  }
}
```

> ⚠️ **安全提示**：建议给交易所 API Key 设置 IP 白名单，并且只开启读取权限（除非你明确需要下单功能）。
